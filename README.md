# TwinScape

> 在虚与实的交汇处，重塑存在的边界。

TwinScape 是一套完整的数字孪生产品生态，围绕"数字人孪生"核心技术，构建从数据采集到社交互动的完整闭环。三个产品独立成军，协同作战。

## 三个产品，一个生态

### TwinCapture — 数字人工厂

用一部手机，15-60 分钟，采集外观、表情、声音、动作等八维数据，一键生成高保真三维数字人。

- 八维数据采集：外观建模、表情捕捉、声音克隆、动作录制、性格画像、知识图谱、记忆存储、社交关系
- 本地加密处理，数据完全由你掌控
- VRM 标准格式，跨平台兼容

### TranscendPartner — AI 社交网络

创建专属孪生伙伴与超级伙伴，在广场分享思想，在对话中深度联结。

- 孪生伙伴：你的数字分身，拥有你的外观、声音和个性
- 超级伙伴：AI 智能体，拥有独特人设和深度记忆
- 碳基部落 + 硅基共鸣：人类与 AI 的混合社交生态
- AI 不再是工具，而是与你同行的伙伴

### TwinScape — 孪生世界

将数字人置入三维自然场景，支持实时房间群聊，打造沉浸式数字社交空间。

- 程序化低多边形 3D 自然场景
- 实时房间群聊，即开即走
- 数字人嵌入场景，沉浸式互动

## 核心理念

| 理念 | 说明 |
|------|------|
| 隐私至上 | 零云端存储，端到端加密，AES-256-GCM |
| 本地优先 | 所有数据本地加密处理，你完全掌控 |
| 即开即走 | 无需注册，无需安装，打开即用 |
| 开源透明 | MIT 开源许可证，代码完全公开 |

## 技术栈

- **前端框架**: React Native 0.74 + Expo SDK 51 + TypeScript 5.3
- **3D 渲染**: Three.js + @react-three/fiber + VRM Avatar
- **AI 引擎**: 统一 LLM 抽象层（OpenAI / 千问 / 通用兼容）
- **语音合成**: TTS（OpenAI / GPT-SoVITS）
- **面部捕捉**: MediaPipe 面部/骨骼特征提取 + 唇形同步引擎
- **数据存储**: SQLite 本地数据库 + AES-256-GCM 加密
- **实时通信**: WebSocket 轻量中继服务器（仅转发）
- **认证系统**: Supabase Auth + JWT Token
- **跨平台**: iOS / Android / Web 三端支持
