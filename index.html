我们用的技术

统一的技术栈确保三个产品无缝协同。从 3D 渲染到 AI 推理，从本地存储到实时通信，每一层都经过精心选型。

前端框架
  • React Native 0.74 + Expo SDK 51
  • TypeScript 5.3 全栈类型安全
  • React Navigation 6 导航
  • iOS / Android / Web 三端跨平台
3D 引擎
  • Three.js 0.164 + @react-three/fiber
  • VRM 标准格式 Avatar 渲染
  • @react-three/drei 辅助工具集
  • 程序化低多边形场景生成
AI 能力
  • 统一 LLM 抽象层（OpenAI/千问/通用兼容）
  • TTS 语音合成（OpenAI/GPT-SoVITS）
  • MediaPipe 面部/骨骼特征提取
  • 唇形同步引擎（80+ 音素映射）
数据与安全
  • SQLite 本地数据库（op-sqlite）
  • AES-256-GCM 文件加密
  • expo-secure-store 安全存储
  • GitHub 私有仓库一键备份
网络与认证
  • WebSocket 轻量中继服务器（仅转发）
  • Supabase Auth 隐私优先认证
  • JWT Token + 自动刷新
  • CDN 简化版数字人分发
开放标准
  • VRM — 3D Avatar 国际标准
  • glTF / SMPL-X — 3D 模型格式
  • ONNX — 语音模型格式
  • MIT 开源许可证
